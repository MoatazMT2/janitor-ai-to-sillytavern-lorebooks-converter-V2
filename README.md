# Lorebook Converter V2

A simple tool to convert Janitor.ai lorebooks to SillyTavern format.

[**Live Demo (GitHub Pages)**](https://moatazmt2.github.io/janitor-ai-to-sillytavern-lorebooks-converter-V2/)

it was originally made by [**hydall**](https://github.com/hydall) all i did is edit it and add a few extra features.

# what changed?

* added the ability to preview your lorebooks and edit them from there, so you can get them ready and use them already at your expectations

* tokens estimator that estimates how much token the lorebook's gonna be.
